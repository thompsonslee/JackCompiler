# Jack Compiler
Jack is a high level OOP language used in the nand2tetris course: https://www.nand2tetris.org/ 

# Instructions

git clone this repository. then run \
cd JackCompiler && sudo ./compile.sh

To Compile a .jack source project, run \
java JackCompiler.Compiler [FILE/DIR]

this will give you .VM file(s). to run these files, you will have to use the online VM Emulator at:
https://nand2tetris.github.io/web-ide/vm
